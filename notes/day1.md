# Day 1 - Nmap Basics

## Commands Used
- nmap 127.0.0.1
- nmap -sV 127.0.0.1
- nmap -p- 127.0.0.1

## What I Learned
- Nmap scans for open ports
- -sV detects service versions
- -p- scans all ports

## Observations
- Port 22 → SSH
- Port 80 → HTTP
